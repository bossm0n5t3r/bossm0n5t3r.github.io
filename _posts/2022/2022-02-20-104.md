간단한 2차원 배열을 만드는 방법이 있다.

```kotlin
fun main() {
    val row = 5
    val col = 4
    val matrix = Array(row) { IntArray(col) }
    matrix.forEach { rows ->
    	println(rows.toList())
    }
}
```
