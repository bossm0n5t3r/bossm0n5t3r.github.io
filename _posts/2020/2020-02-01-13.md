# Basic Git commands

<img src="/assets/images/posts/logos/Git-Logo-1788C.png" width="600">

## git config

### Usage

```bash
  $ git config --global user.name "[name]"
  $ git config --global user.email "[email]"
```

- git 에게 사용자가 누구인지 설정

## git init

### Usage

```bash
  $ git init
```

- 로컬 repository 생성
