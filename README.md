# The Archive

## bossm0n5t3r's blog

This repository is based on [Simple Texture Jekyll Theme](https://github.com/yizeng/jekyll-theme-simple-texture)

## Usage

> #### Prerequisite
>
> - ruby

```sh
  $ gem install jekyll bundler

  # cd blog repository

  $ bundle install
  $ bundle exec jekyll serve

  # Hack away at <http://localhost:4000>!
```
