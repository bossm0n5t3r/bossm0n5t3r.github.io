# [The Archive](https://bossm0n5t3r.github.io/)

This repository is based on [Sidey](https://github.com/ronv/sidey)

## Usage

> #### Prerequisite
>
> - ruby

```sh
  $ sudo gem install jekyll bundler

  # cd blog repository

  $ bundle install
  $ bundle exec jekyll serve

  # Hack away at <http://localhost:4000>!
```
