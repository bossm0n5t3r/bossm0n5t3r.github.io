This repository is based on [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy)

## Usage

> #### Prerequisite
>
> - ruby

```sh
  $ gem install jekyll bundler

  # cd blog repository

  $ bundle install
  $ bundle exec jekyll serve

  # Hack away at <http://localhost:4000>!
```
